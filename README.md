# README – Análisis de Datos con Pandas y Matplotlib

## Descripción

Este proyecto realiza el análisis de datos provenientes de un archivo CSV utilizando Python en JupyterLab. Se emplea la librería Pandas para procesar información y Matplotlib para generar gráficas estadísticas.

## Tecnologías utilizadas

* Python
* JupyterLab
* Pandas
* Matplotlib

## Funciones del programa

* Lectura de archivos CSV.
* Análisis estadístico de datos.
* Cálculo de promedio, máximo y mínimo.
* Generación de gráficas de líneas y barras.
* Exportación de resultados a un nuevo archivo CSV.

## Archivo utilizado

```plaintext
ventas.csv
```

## Ejecución del programa

Instalar librerías:

```python
pip install pandas
pip install matplotlib
```

Ejecutar el archivo Python en JupyterLab.

## Resultados obtenidos

El programa muestra:

* Estadísticas descriptivas.
* Gráficas de ventas y gastos.
* Comparación visual de datos.
* Archivo CSV con resultados procesados.
